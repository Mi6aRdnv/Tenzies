# Tenzies

[Scrimba learning project](https://scrimba.com/learn-react-c0e/~03pz)

Tenzies is a small interactive dice game implemented as a local development project. The goal is to get all dice showing the same number by rolling and "holding" dice until they match.

GitHub Pages - https://mi6ardnv.github.io/Tenzies/

## How to play

- Click "Roll" to roll all not held dice.
- Click a die to toggle hold on that die.
- Continue until all dice show the same number to win.
